Hands-on Aktiviteter

Att lära sig om AI handlar inte bara om teori, utan också om att få praktisk erfarenhet. Dessa övningar är utformade för att ge dig en känsla för hur AI-system fungerar i praktiken, även utan avancerade programmeringskunskaper. Vi kommer att använda webbaserade verktyg och enkla plattformar för att utforska grundläggande AI-koncept.

Bildklassificering med Teachable Machine
Använd Google Teachable Machine för att träna en enkel bildklassificerare. Du kan lära den att känna igen olika objekt, ansiktsuttryck eller gester. Experimentera med olika träningsdata och se hur det påverkar AI:ns prestanda.
Enkel Prediktion med Regression
Använd ett enkelt online-verktyg för linjär regression (t.ex. Google Sheets med trendlinjer eller en enkel webbapp) för att förutsäga framtida värden baserat på historisk data. Exempel kan vara att förutsäga huspriser baserat på storlek, eller försäljning baserat på marknadsföringsutgifter.
Bygg en Chatbot med Dialogflow (eller liknande)
Skapa en grundläggande chatbot som kan svara på fördefinierade frågor. Fokusera på att designa intentioner och entiteter för att chatboten ska förstå användarens frågor och ge relevanta svar.
Spela mot en AI i Tic-Tac-Toe
Många enkla AI-agenter kan spela Tic-Tac-Toe. Försök att förstå strategin bakom AI:ns drag. Kan du hitta ett sätt att alltid vinna eller spela oavgjort mot den? Hur skiljer sig AI:ns "tänkande" från ditt eget?

Reflektionsfrågor

  • Vilka utmaningar stötte du på när du tränade din AI i bildklassificering?
  • Hur påverkade kvaliteten på din träningsdata AI:ns förmåga att göra korrekta förutsägelser?
  • Vad var det mest överraskande du lärde dig om AI genom dessa praktiska övningar?
